# **App Name**: Devfolio Showcase

## Core Features:

- Project Gallery: Showcase of development projects with descriptions, images, and links to live demos or GitHub repositories.
- About Me Section: Detailed information about the developer, including skills, experience, education, and a personal statement.
- Contact Form: A form for visitors to send messages directly to the developer.
- Skills Visualization: A visual representation of the developer's skills, such as a bar graph or word cloud.
- Personalized Recommendations: AI powered module to give suggestions and recommendation for technologies or career direction. LLM uses your projects and skillsets to provide meaningful insights

## Style Guidelines:

- Primary color: Deep navy blue (#21295C) to convey professionalism and technical expertise.
- Background color: Light gray (#E7E7EB) for a clean and modern feel.
- Accent color: Electric purple (#BE31E7) to highlight important elements and create visual interest.
- Body font: 'Inter', a sans-serif font for clean and modern readability.
- Headline font: 'Space Grotesk', a sans-serif font for impactful headings.
- Use simple, line-based icons from a library like Font Awesome or Remix Icon for navigation and key features.
- Utilize a grid-based layout with clear sections and ample white space for a balanced and easy-to-navigate design.
- Subtle transition effects when scrolling or hovering over project cards to add interactivity.
